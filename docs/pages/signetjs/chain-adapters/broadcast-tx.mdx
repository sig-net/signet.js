# broadcastTx

The `broadcastTx` method broadcasts a signed transaction.

## Usage

```ts twoslash
// [!include ~/snippets/code/chains.ts]
// ---cut---
const txHash = await evmChain.broadcastTx('0x02f8...')
```

## Parameters

| Parameter      | Type     | Description                       |
| -------------- | -------- | --------------------------------- |
| `txSerialized` | `string` | The serialized signed transaction |

## Returns

| Type     | Description          |
| -------- | -------------------- |
| `string` | The transaction hash |

## Working Examples

- [EVM](https://github.com/sig-net/signet.js/blob/main/tests/unit/evm/EVM.test.ts) — broadcast an EIP-1559 transaction
- [Bitcoin](https://github.com/sig-net/signet.js/blob/main/tests/unit/btc/Bitcoin.test.ts) — broadcast single and multi-input transactions
- [Cosmos](https://github.com/sig-net/signet.js/blob/main/tests/unit/cosmos/Cosmos.test.ts) — broadcast a bank send transaction
